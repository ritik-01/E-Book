Operators in java

Operator in java is a symbol that is used to perform operations. For example: +, -, *, / etc.

 

There are many types of operators in java which are given below:

  • Unary Operator,
  • Arithmetic Operator,
  • Shift Operator,
  • Relational Operator,
  • Bitwise Operator,
  • Logical Operator,
  • Ternary Operator and
  • Assignment Operator.
